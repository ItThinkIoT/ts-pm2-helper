# PM2 Helper

PM2 Helper for Typescript

## Install

- Yarn
```shell
yarn add @itthink/ts-pm2-helper
```

- NPM
```shell
npm install @itthink/ts-pm2-helper
```

## Features

- Load Ecosystem Variables: load variables from ecosystem file to `process.env`.
    - Function name: `loadEcosystemVariables()`
